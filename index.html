Chess Engine

A chess engine built in C/C++. It features a fully playable chess board and a chess algorithm to play against. I made the algorithm from scratch using my own move generation and something called negamax, an optimization of minimax. The engine has all proper chess rules implemented like castling/en passant/promoting/ and draw by repitition. For the graphics to display the board I used a library called RayLib. I estimate the algorithm to play at a level of approximately 1000 elo.

Lyric Analyzer

Ever wondered what your favorite artists meant by a specific lyric? This site utilizes new technologies and different API's to summarize and analyze songs. Users can search for any song and my application uses Genius API to get song meta data. Finally Google's Gemini is used to summarize the deeper meaning of the song and analyze specific lyrics the user selects. The backend uses Node.js while the front end is plain HTML/CSS/JS. I implemented the Spotify API into this as well to automatically detect the currently playing song.

Audio Visualizer

This is a web application that can generate mulitple visualizers to any song! The javascript uses the Canvas API as well as the Web Audio API to display frequencies on screen. There are plenty of controls like coloring, a custom audio controller with volume slider, as well as 5 different visualization algorithms.

Skateboarding Simulator

Game developed in C# using the Unity game engine. This virtual skateboarding simulation allows the player to cruise freely and express their creativity. Every trick imaginable can be performed in combination with multiple objects to grind/slide on. Many of the skateboarding physics I implemented through code, teaching me a lot about C# and game engines in general.

2D Platformer

A simple platformer built from scratch in Java. I used the built in JPanel and JFrame for graphics aswell as the system clock to get a consistent game loop. The game is infinitely long with the score increasing as the player jumps higher and avoids the snakes. All of the sprites are drawn by me using a pixel art editor. This project taught me a lot about Java and Object Oriented Programming.

2D Platformer

A simple platformer built from scratch in Java. I used the built in JPanel and JFrame for graphics aswell as the system clock to get a consistent game loop. The game is infinitely long with the score increasing as the player jumps higher and avoids the snakes. All of the sprites are drawn by me using a pixel art editor. This project taught me a lot about Java and Object Oriented Programming.
